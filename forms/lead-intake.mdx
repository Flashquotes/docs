## Overview

The Lead Intake Form is optimized for high conversion rates, collecting only essential information needed to generate accurate quotes. This form automatically generates when you create your account and can be embedded on your website.

## Form Structure

<AccordionGroup>
  <Accordion icon="page" title="Page-by-Page Breakdown">
    The lead intake form is optimized with minimal, essential questions:

    1. **Service Details**
       - Service length
       - Guest count
       - Dynamic staff calculation

    2. **Location**
       - Event address
       - Venue details

    3. **Timing**
       - Event date
       - Service start time
       - Number of event days

    4. **Event Type**
       - Category selection
       - Pricing impact factors

    5. **Customization**
       - Add-ons selection
       - Special requirements

    6. **Additional Information**
       - Event details
       - Referral source

    7. **Contact Details**
       - First name
       - Last name
       - Email
       - Phone number

  </Accordion>

  <Accordion icon="bolt" title="Instant Quoting">
    Enable automatic quote generation and presentation:
    
    1. Click the "Instant Pricing" button in form settings
    2. Toggle "Quoting Mode" to "Instant Quoting"
    3. Configure default service and pricing rules
    
    When enabled:
    - Quotes are generated instantly upon form submission
    - Clients are redirected to the quote presentation page
    - "Book Now" option is immediately available
    
    <Note>
      Instant Quoting is a Pro plan feature that enables clients to inquire, receive quotes, and book in a single session.
    </Note>
  </Accordion>

## Website Integration

<AccordionGroup>
  <Accordion icon="code" title="Form Embedding">
    To embed the lead intake form: 1. Click the `Embed Form` button 2. Select
    your website builder 3. Copy the provided code 4. Paste into your website's
    contact form location
    <Info>
      Flashquotes forms are compatible with virtually all website platforms. See
      our [Platform Compatibility](/forms/compatibility) guide for details on
      embedding in specific platforms.
    </Info>
    <Warning>
      Wix websites require specific embedding code. Select 'Wix' from the
      dropdown for correct code.
    </Warning>
  </Accordion>
</AccordionGroup>
