## Overview

Resources in Flashquotes represent your physical business assets - from core service equipment to transportation vehicles. Proper resource management ensures accurate availability tracking and efficient event delivery.

## Resource Types

  <Accordion icon="truck" title="Core Service Assets">

    - Service trailers

## Resource Management

<AccordionGroup>
  <Accordion icon="plus" title="Adding Resources">
    To add a new resource:
    1. Click `Add Resource` in the top right
    2. Enter resource details:
       - Name (e.g., "Cart One", "White Cart")
       - Type selection
       - Location assignment
       - Availability status
    
    <Tip>
      Consider using themed names for resources to make them memorable and distinctive.
    </Tip>
  </Accordion>

  <Accordion icon="pen" title="Resource Details">
    Configure client-facing information:
    - Space requirements
    - Power needs
    - Event type suitability
    - Special features
    - Setup requirements
  </Accordion>

  <Accordion icon="image" title="Resource Images">
    Upload photos to:
    - Help clients make selections
    - Provide visual reference
    - Enhance booking form experience
    - Document equipment condition
  </Accordion>
</AccordionGroup>
