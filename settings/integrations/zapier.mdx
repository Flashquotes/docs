## Overview

The Zapier integration allows you to connect Flashquotes with thousands of other apps and create automated workflows. Available for Pro tier subscribers, this integration helps streamline your operations through custom automations.

## Setup Process

<AccordionGroup>
  <Accordion icon="key" title="API Key Generation">
    Get your API credentials:
    1. Navigate to Integrations settings
    2. Select Zapier
    3. Generate API key
    4. Copy key for Zapier setup
    
    <Warning>
      Keep your API key secure and never share it publicly.
    </Warning>
  </Accordion>

  <Accordion icon="plug" title="Zapier Connection">
    Connect to Zapier:
    1. Visit Zapier.com
    2. Search for Flashquotes
    3. Add to your Zapier account
    4. Enter your API key
    
    <Note>
      Requires an active Zapier account and Pro tier Flashquotes subscription.
    </Note>
  </Accordion>
</AccordionGroup>
