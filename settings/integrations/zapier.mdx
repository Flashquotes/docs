Generate an API key from Settings > Integrations to connect Flashquotes with thousands of apps.

## What it does

The Zapier integration allows you to connect Flashquotes with thousands of other apps and create automated workflows. Available for Pro tier subscribers, this integration helps streamline your operations through custom automations.

## Get your API key

<Steps>
  <Step title="Go to integrations">
    Navigate to Settings > Integrations
  </Step>
  <Step title="Find the API & Zapier card">
    On **Pro**, you'll see a Zapier card (with a "Generate API key" action) and a separate full-API card promoting the Scale upgrade. On **Scale**, both surfaces are merged into a single "API & Zapier" card — the same key works for Zapier and for the full [REST API](/api-reference/introduction). Click "Generate API key".
  </Step>
  <Step title="Copy API key">
    Copy the generated key for use in Zapier
  </Step>
  <Step title="Connect in Zapier">
    Go to Zapier.com, search for Flashquotes, and enter your API key
  </Step>
</Steps>

<Warning>
Keep your API key secure and never share it publicly.
</Warning>

<Note>
Requires an active Zapier account and a Pro tier Flashquotes subscription (or higher). The full REST API requires Scale — see [API plan access](/api-reference/introduction#plan-access).
</Note>

- Lead form submitted
- Lead status changes
- Quote generated

**Event triggers**:
- New booking created
- Event status changes
- Timeline updates
- Resource assignments

**Invoice triggers**:
- New invoice created
- Payment received
- Invoice status changes
- Refund processed
