## Overview

Task settings allow you to manage templates for both administrative tasks and event brief tasks. These templates automatically populate when creating new events, ensuring consistency in your operations.

## Task Templates

<AccordionGroup>
  <Accordion icon="list-check" title="Admin Tasks">
    Configure administrative task templates:
    - Pre-event preparations
    - Client communications
    - Resource management
    - Post-event follow-ups
    
    <Tip>
      Create comprehensive task lists to ensure nothing is overlooked during event preparation.
    </Tip>
  </Accordion>

  <Accordion icon="file-lines" title="Event Brief Tasks">
    Manage event brief sections:
    - Task name and description
    - Section assignment
    - Task ordering
    - Required actions
    
    <Note>
      As of January 2025, you can use Markdown formatting in task descriptions for better organization and linking.
    </Note>
  </Accordion>
</AccordionGroup>

## Markdown Support

<AccordionGroup>
  <Accordion icon="markdown" title="Formatting Options">
    Use Markdown to enhance task descriptions:
    ```markdown
    - Create links: [Gallery](https://example.com/gallery)
    - Add emphasis: **Important** or *Note*
    - Create lists: 1. First 2. Second
    - Add headings: # Main Point
    ```
    
    <Info>
      Markdown formatting helps create clear, structured task instructions.
    </Info>
  </Accordion>

  <Accordion icon="link" title="Link Integration">
    Add useful links in tasks:
    - Equipment manuals
    - Setup guides
    - Training videos
    - Reference documents
  </Accordion>
</AccordionGroup>

1. Create clear, actionable tasks
2. Use consistent formatting
3. Keep instructions concise
4. Review and update regularly
5. Maintain logical task order
