Helper functions provide even more power and flexibility when you use shortcodes. The general syntax for helper functions is `{{function_name variable_1 variable_2...}}` where the _function_name_ value tells the system which function to trigger.

Sometimes more advanced functions will have an open and closing tag which are formatted as:

- Opening tag: `{{#function_name}}`
- Closing tag: `{{/function_name}}`

<Info>
	For example, the [If Conditions](/workflows/shortcodes/helpers/if-else) us
	opening and closing tag syntax
</Info>

## Available Functions

- [Default Values](/workflows/shortcodes/helpers/default) - Specify default fallback values when fields are empty
- [If Conditions](/workflows/shortcodes/helpers/if-else) - Return something **if** a value is not empty
- [Unless Conditions](/workflows/shortcodes/helpers/unless) - Return something **unless** a value is not empty
